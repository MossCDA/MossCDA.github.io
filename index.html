PROJET 01
CASSANDRE
OPEN SOURCE SIGNAL INTELLIGENCE

Système de détection de signaux faibles sur sources ouvertes. Collecte automatisée de plus de 40 sources (presse nationale, régionale, forums, réseaux sociaux), classification par Mistral 7B en local, cartographie Leaflet des tensions mondiales et algorithmes de détection de corrélations inter-thèmes.

Origine : détection manuelle du signal COVID-19 sur un forum de jeux vidéo en janvier 2020, plusieurs semaines avant les alertes officielles. CASSANDRE automatise cette méthode.

PROJET 02
VOXREPORT
RAPPORT VOCAL NUMÉRIQUE

Application web MVC PHP de création de rapports par dictée vocale. Transcription via Web Speech API, signature électronique sur canvas, génération PDF horodatée et workflow de validation multi-rôles (citoyen / agent / administrateur).

Cible opérationnelle : police municipale, sécurité privée, inspection du travail — tout contexte nécessitant un rapport rapide sans clavier.

02 · PROFIL

Reconversion,
pas débutant.

Parcours atypique : militaire (13e RDP), télécoms, logistique, usinage. Reconversion en développement d'applications — RNCP niveau 6 en cours — avec une orientation délibérée vers les outils à usage opérationnel et institutionnel.

L'intérêt pour l'OSINT et la détection de signaux faibles précède largement le CDA : détection manuelle du signal COVID-19 sur sources ouvertes en janvier 2020, plusieurs semaines avant les alertes officielles. CASSANDRE est l'automatisation de cette méthode.

Approche : souveraineté par défaut — IA locale, données locales, déploiement hors-ligne possible. Les outils que je construis sont conçus pour des environnements où la confidentialité n'est pas optionnelle.
